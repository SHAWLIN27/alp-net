# ALP_Net Architecture

ALP_Net uses Obsidian as an authoring vault, a Python ingestion pipeline, NetworkX+JSON as the graph store, ChromaDB as the vector store, FastAPI as the backend, and React+Cytoscape.js as the visualization frontend.
