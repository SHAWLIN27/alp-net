# 1. Overview / 概述

**English:**
Projectile motion graphs are essential tools for visualizing and analyzing the two-dimensional motion of objects under gravity. This sub-topic focuses on interpreting and constructing graphs that represent the horizontal and vertical components of displacement, velocity, and acceleration separately. Unlike simple linear motion, projectile motion requires understanding how the horizontal component remains constant while the vertical component changes due to gravity. These graphs are crucial for solving complex problems without relying solely on algebraic equations, and they form a bridge between [[Horizontal and Vertical Components]] and concepts like [[Time of Flight and Range]]. Mastery of these graphs is a key skill for A-Level Physics, as they appear frequently in both multiple-choice and structured questions.

**中文:**
抛体运动图像是可视化和分析物体在重力作用下二维运动的重要工具。本子知识点专注于解读和绘制代表位移、速度和加速度的水平与垂直分量分别的图像。与简单的直线运动不同，抛体运动需要理解水平分量保持不变，而垂直分量因重力而变化。这些图像对于不单纯依赖代数方程解决复杂问题至关重要，它们构成了连接[[水平与垂直分量]]与[[飞行时间与射程]]等概念的桥梁。掌握这些图像是A-Level物理的关键技能，因为它们经常出现在选择题和结构化题目中。

# 3. Core Definitions / 核心定义

| Term (EN/CN) | Definition (EN) | Definition (CN) | Common Mistakes / 常见错误 |
|--------------|-----------------|-----------------|---------------------------|
| **Horizontal Component Graph** / 水平分量图像 | A graph showing the motion of a projectile in the x-direction, where acceleration is zero and velocity is constant. | 显示抛体在x方向运动的图像，其中加速度为零，速度恒定。 | Thinking the horizontal velocity changes; it remains constant (ignoring air resistance). |
| **Vertical Component Graph** / 垂直分量图像 | A graph showing the motion of a projectile in the y-direction, where acceleration is constant (g = 9.81 m/s² downward). | 显示抛体在y方向运动的图像，其中加速度恒定（g = 9.81 m/s² 向下）。 | Forgetting that acceleration is always downward, even at the peak. |
| **Parabolic Displacement-Time Graph** / 抛物线位移-时间图像 | The curved shape of the vertical displacement vs. time graph, resulting from constant acceleration. | 垂直位移与时间图像的弯曲形状，由恒定加速度引起。 | Confusing this with the horizontal displacement-time graph, which is linear. |
| **Gradient** / 斜率 | The slope of a graph, representing velocity (on displacement-time) or acceleration (on velocity-time). | 图像的斜率，代表速度（在位移-时间图上）或加速度（在速度-时间图上）。 | Forgetting that gradient = Δy/Δx; misreading scales. |
| **Area Under Graph** / 曲线下面积 | The area between the graph line and the time axis, representing displacement (on velocity-time) or change in velocity (on acceleration-time). | 图像线与时间轴之间的面积，代表位移（在速度-时间图上）或速度变化（在加速度-时间图上）。 | Using the wrong formula for area (e.g., triangle vs. trapezium). |

# 4. Key Concepts Explained / 关键概念详解

## 4.1 Independence of Horizontal and Vertical Motion / 水平与垂直运动的独立性

### Explanation / 解释
**English:** The fundamental principle of projectile motion is that the horizontal and vertical components are independent of each other. This means you can analyze the motion in the x-direction and y-direction separately, using different graphs for each. The horizontal component has zero acceleration (ignoring air resistance), so its velocity-time graph is a horizontal line, and its displacement-time graph is a straight line with constant gradient. The vertical component has constant acceleration due to gravity (g = 9.81 m/s² downward), so its velocity-time graph is a straight line with gradient g, and its displacement-time graph is a parabola. This independence is derived from [[Scalars and Vectors]] and the [[Equations of Motion (SUVAT)]] applied separately to each component.

**中文:** 抛体运动的基本原理是水平与垂直分量相互独立。这意味着你可以分别分析x方向和y方向的运动，为每个方向使用不同的图像。水平分量加速度为零（忽略空气阻力），因此其速度-时间图像是一条水平线，位移-时间图像是一条斜率恒定的直线。垂直分量因重力而有恒定加速度（g = 9.81 m/s² 向下），因此其速度-时间图像是一条斜率为g的直线，位移-时间图像是一条抛物线。这种独立性源于[[标量与向量]]以及分别应用于每个分量的[[运动方程（SUVAT）]]。

### Physical Meaning / 物理意义
**English:** Physically, this means that the time it takes for a projectile to reach its maximum height depends only on the vertical component of initial velocity, not the horizontal component. Similarly, the horizontal range depends on both the horizontal velocity and the time of flight (which is determined vertically). Graphs help visualize this: the vertical velocity-time graph crosses zero at the peak, while the horizontal velocity-time graph remains unchanged.

**中文:** 物理上，这意味着抛体达到最大高度所需的时间仅取决于初速度的垂直分量，而非水平分量。同样，水平射程取决于水平速度和飞行时间（由垂直方向决定）。图像有助于可视化这一点：垂直速度-时间图像在最高点穿过零，而水平速度-时间图像保持不变。

### Common Misconceptions / 常见误区
- **English:**
  - Thinking the horizontal velocity decreases over time (it doesn't, without air resistance).
  - Believing the vertical velocity is zero at the start (it's only zero if launched horizontally).
  - Confusing the parabolic shape of vertical displacement with the path of the projectile (the path is also parabolic, but the graph is different).
- **中文:**
  - 认为水平速度随时间减小（没有空气阻力时不会）。
  - 认为垂直速度在开始时为零（只有水平发射时才为零）。
  - 将垂直位移的抛物线形状与抛体的轨迹混淆（轨迹也是抛物线，但图像不同）。

### Exam Tips / 考试提示
- **English:** Always sketch the horizontal and vertical graphs separately. Label axes clearly with units. For vertical motion, remember that acceleration is always downward, so the velocity-time graph has a negative gradient if upward is positive.
- **中文:** 始终分别绘制水平和垂直图像。清晰标注坐标轴及单位。对于垂直运动，记住加速度始终向下，因此如果向上为正，速度-时间图像具有负斜率。

## 4.2 Key Graphs for Projectile Motion / 抛体运动的关键图像

### Explanation / 解释
**English:** There are six key graphs to master: horizontal displacement-time, horizontal velocity-time, horizontal acceleration-time, vertical displacement-time, vertical velocity-time, and vertical acceleration-time. Each has a distinct shape that reveals information about the motion. For example, the vertical velocity-time graph is a straight line with gradient -g (if upward is positive), crossing zero at the peak. The horizontal displacement-time graph is a straight line with gradient equal to the initial horizontal velocity. The vertical displacement-time graph is a parabola, symmetric about the peak time.

**中文:** 需要掌握六个关键图像：水平位移-时间、水平速度-时间、水平加速度-时间、垂直位移-时间、垂直速度-时间和垂直加速度-时间。每个都有独特的形状，揭示运动信息。例如，垂直速度-时间图像是一条斜率为-g（如果向上为正）的直线，在最高点穿过零。水平位移-时间图像是一条斜率等于初始水平速度的直线。垂直位移-时间图像是一条抛物线，关于最高点时间对称。

### Physical Meaning / 物理意义
**English:** The gradient of the displacement-time graph gives velocity. The gradient of the velocity-time graph gives acceleration. The area under the velocity-time graph gives displacement. For vertical motion, the area under the velocity-time graph from t=0 to the peak gives the maximum height. The total area (positive minus negative) from t=0 to landing gives zero net vertical displacement (if launch and landing heights are equal).

**中文:** 位移-时间图像的斜率给出速度。速度-时间图像的斜率给出加速度。速度-时间图像下的面积给出位移。对于垂直运动，速度-时间图像下从t=0到最高点的面积给出最大高度。从t=0到落地的总面积（正面积减去负面积）给出净垂直位移为零（如果发射和落地高度相等）。

### Common Misconceptions / 常见误区
- **English:**
  - Thinking the vertical displacement-time graph is a straight line (it's parabolic due to constant acceleration).
  - Believing the horizontal acceleration-time graph has a non-zero value (it's zero).
  - Misinterpreting the area under the vertical velocity-time graph as the range (it's the vertical displacement).
- **中文:**
  - 认为垂直位移-时间图像是直线（由于恒定加速度，它是抛物线）。
  - 认为水平加速度-时间图像有非零值（它是零）。
  - 将垂直速度-时间图像下的面积误解为射程（它是垂直位移）。

### Exam Tips / 考试提示
- **English:** When sketching graphs, always consider the initial conditions. If the projectile is launched from ground level, the vertical displacement starts at zero, rises to a maximum, and returns to zero. If launched from a height, the final displacement is negative. Use the [[Equations of Motion (SUVAT)]] to find key points like the time to peak and the maximum height.
- **中文:** 绘制图像时，始终考虑初始条件。如果抛体从地面发射，垂直位移从零开始，上升到最大值，然后回到零。如果从高度发射，最终位移为负。使用[[运动方程（SUVAT）]]找到关键点，如到达最高点的时间和最大高度。

# 5. Essential Equations / 核心公式

## 5.1 Horizontal Motion Equations / 水平运动方程

$$ s_x = u_x t = (u \cos \theta) t $$

$$ v_x = u_x = u \cos \theta $$

$$ a_x = 0 $$

| Symbol (符号) | Meaning (EN) | Meaning (CN) | Unit (单位) |
|--------------|-------------|-------------|------------|
| $s_x$ | Horizontal displacement | 水平位移 | m |
| $u_x$ | Initial horizontal velocity | 初始水平速度 | m s⁻¹ |
| $u$ | Initial velocity (magnitude) | 初速度（大小） | m s⁻¹ |
| $\theta$ | Launch angle from horizontal | 发射角（与水平方向） | ° or rad |
| $t$ | Time | 时间 | s |
| $v_x$ | Horizontal velocity at time t | t时刻的水平速度 | m s⁻¹ |
| $a_x$ | Horizontal acceleration | 水平加速度 | m s⁻² |

**Derivation / 推导:** From [[Equations of Motion (SUVAT)]] with $a_x = 0$, $v_x = u_x + a_x t = u_x$, and $s_x = u_x t + \frac{1}{2} a_x t^2 = u_x t$.

**Conditions / 适用条件:** No air resistance; constant horizontal velocity.

**Limitations / 局限性:** Does not account for air resistance or varying gravity.

## 5.2 Vertical Motion Equations / 垂直运动方程

$$ s_y = u_y t - \frac{1}{2} g t^2 = (u \sin \theta) t - \frac{1}{2} g t^2 $$

$$ v_y = u_y - g t = u \sin \theta - g t $$

$$ a_y = -g $$

| Symbol (符号) | Meaning (EN) | Meaning (CN) | Unit (单位) |
|--------------|-------------|-------------|------------|
| $s_y$ | Vertical displacement | 垂直位移 | m |
| $u_y$ | Initial vertical velocity | 初始垂直速度 | m s⁻¹ |
| $g$ | Acceleration due to gravity (9.81 m s⁻²) | 重力加速度 (9.81 m s⁻²) | m s⁻² |
| $v_y$ | Vertical velocity at time t | t时刻的垂直速度 | m s⁻¹ |
| $a_y$ | Vertical acceleration | 垂直加速度 | m s⁻² |

**Derivation / 推导:** From [[Equations of Motion (SUVAT)]] with $a_y = -g$, $v_y = u_y + a_y t = u_y - g t$, and $s_y = u_y t + \frac{1}{2} a_y t^2 = u_y t - \frac{1}{2} g t^2$.

**Conditions / 适用条件:** Constant gravitational field; no air resistance; upward direction taken as positive.

**Limitations / 局限性:** Assumes g is constant; does not account for air resistance.

# 6. Graphs and Relationships / 图表与关系

## 6.1 Horizontal Displacement-Time Graph / 水平位移-时间图像

### Axes / 坐标轴
- **X-axis:** Time (t / s) / 时间 (t / s)
- **Y-axis:** Horizontal displacement (s_x / m) / 水平位移 (s_x / m)

### Shape / 形状
**English:** A straight line with positive gradient (if launched with positive horizontal velocity). The gradient is constant and equal to the initial horizontal velocity $u_x = u \cos \theta$.

**中文:** 一条具有正斜率的直线（如果以正水平速度发射）。斜率恒定，等于初始水平速度 $u_x = u \cos \theta$。

### Gradient Meaning / 斜率含义
**English:** The gradient represents the horizontal velocity, which is constant.

**中文:** 斜率代表水平速度，是恒定的。

### Area Meaning / 面积含义
**English:** Not typically used for this graph; the area under a displacement-time graph has no physical meaning.

**中文:** 通常不使用此图像的面积；位移-时间图像下的面积没有物理意义。

### Exam Interpretation / 考试解读
**English:** If the graph is a straight line, horizontal velocity is constant. If it curves, there is horizontal acceleration (e.g., due to air resistance). The steeper the line, the greater the horizontal velocity.

**中文:** 如果图像是直线，水平速度恒定。如果弯曲，存在水平加速度（例如由于空气阻力）。线越陡，水平速度越大。

---

## 6.2 Horizontal Velocity-Time Graph / 水平速度-时间图像

### Axes / 坐标轴
- **X-axis:** Time (t / s) / 时间 (t / s)
- **Y-axis:** Horizontal velocity (v_x / m s⁻¹) / 水平速度 (v_x / m s⁻¹)

### Shape / 形状
**English:** A horizontal line at $v_x = u \cos \theta$, indicating constant velocity.

**中文:** 一条在 $v_x = u \cos \theta$ 处的水平线，表示速度恒定。

### Gradient Meaning / 斜率含义
**English:** The gradient represents horizontal acceleration, which is zero.

**中文:** 斜率代表水平加速度，为零。

### Area Meaning / 面积含义
**English:** The area under the graph between t=0 and t=T gives the horizontal displacement during that time. Area = $v_x \times T = (u \cos \theta) T$.

**中文:** 图像下从t=0到t=T的面积给出该时间段内的水平位移。面积 = $v_x \times T = (u \cos \theta) T$。

### Exam Interpretation / 考试解读
**English:** A horizontal line confirms no horizontal forces (ignoring air resistance). The area under this graph is often used to find the range when the time of flight is known.

**中文:** 水平线确认没有水平力（忽略空气阻力）。此图像下的面积常用于在已知飞行时间时求射程。

---

## 6.3 Vertical Displacement-Time Graph / 垂直位移-时间图像

### Axes / 坐标轴
- **X-axis:** Time (t / s) / 时间 (t / s)
- **Y-axis:** Vertical displacement (s_y / m) / 垂直位移 (s_y / m)

### Shape / 形状
**English:** A parabola opening downward (if upward is positive). It starts at zero (if launched from ground), rises to a maximum at the peak time $t_{peak} = \frac{u \sin \theta}{g}$, then returns to zero at the time of flight $T = \frac{2u \sin \theta}{g}$.

**中文:** 一条开口向下的抛物线（如果向上为正）。从零开始（如果从地面发射），在最高点时间 $t_{peak} = \frac{u \sin \theta}{g}$ 上升到最大值，然后在飞行时间 $T = \frac{2u \sin \theta}{g}$ 回到零。

### Gradient Meaning / 斜率含义
**English:** The gradient at any point represents the vertical velocity at that time. At the peak, the gradient is zero (vertical velocity = 0).

**中文:** 任意点的斜率代表该时刻的垂直速度。在最高点，斜率为零（垂直速度 = 0）。

### Area Meaning / 面积含义
**English:** Not typically used for this graph.

**中文:** 通常不使用此图像的面积。

### Exam Interpretation / 考试解读
**English:** The symmetry of the parabola indicates that the time to rise equals the time to fall (if launch and landing heights are equal). The maximum height can be read from the peak of the graph. If launched from a height, the graph does not return to zero.

**中文:** 抛物线的对称性表明上升时间等于下降时间（如果发射和落地高度相等）。最大高度可以从图像的峰值读取。如果从高度发射，图像不会回到零。

---

## 6.4 Vertical Velocity-Time Graph / 垂直速度-时间图像

### Axes / 坐标轴
- **X-axis:** Time (t / s) / 时间 (t / s)
- **Y-axis:** Vertical velocity (v_y / m s⁻¹) / 垂直速度 (v_y / m s⁻¹)

### Shape / 形状
**English:** A straight line with negative gradient (if upward is positive). It starts at $v_y = u \sin \theta$, crosses zero at $t_{peak} = \frac{u \sin \theta}{g}$, and reaches $-u \sin \theta$ at $t = T = \frac{2u \sin \theta}{g}$.

**中文:** 一条具有负斜率的直线（如果向上为正）。从 $v_y = u \sin \theta$ 开始，在 $t_{peak} = \frac{u \sin \theta}{g}$ 穿过零，在 $t = T = \frac{2u \sin \theta}{g}$ 达到 $-u \sin \theta$。

### Gradient Meaning / 斜率含义
**English:** The gradient represents vertical acceleration, which is constant at $-g = -9.81 \text{ m s}^{-2}$.

**中文:** 斜率代表垂直加速度，恒定在 $-g = -9.81 \text{ m s}^{-2}$。

### Area Meaning / 面积含义
**English:** The area under the graph gives vertical displacement. The area above the time axis (from t=0 to t_peak) gives the maximum height. The area below the time axis (from t_peak to T) gives the negative displacement during descent. The net area (positive minus negative) is zero if launch and landing heights are equal.

**中文:** 图像下的面积给出垂直位移。时间轴上方的面积（从t=0到t_peak）给出最大高度。时间轴下方的面积（从t_peak到T）给出下降过程中的负位移。如果发射和落地高度相等，净面积（正减负）为零。

### Exam Interpretation / 考试解读
**English:** This is the most important graph for projectile motion. The time when the line crosses the t-axis is the time to reach maximum height. The area of the triangle above the axis gives the maximum height. The total area (triangle above minus triangle below) gives net vertical displacement.

**中文:** 这是抛体运动最重要的图像。线与t轴相交的时间是达到最大高度的时间。轴上方三角形的面积给出最大高度。总面积（上方三角形减去下方三角形）给出净垂直位移。

---

## 6.5 Vertical Acceleration-Time Graph / 垂直加速度-时间图像

### Axes / 坐标轴
- **X-axis:** Time (t / s) / 时间 (t / s)
- **Y-axis:** Vertical acceleration (a_y / m s⁻²) / 垂直加速度 (a_y / m s⁻²)

### Shape / 形状
**English:** A horizontal line at $a_y = -g = -9.81 \text{ m s}^{-2}$, indicating constant acceleration downward.

**中文:** 一条在 $a_y = -g = -9.81 \text{ m s}^{-2}$ 处的水平线，表示向下的恒定加速度。

### Gradient Meaning / 斜率含义
**English:** The gradient is zero (constant acceleration).

**中文:** 斜率为零（恒定加速度）。

### Area Meaning / 面积含义
**English:** The area under the graph gives the change in vertical velocity. Area = $(-g) \times T = -gT$, which equals the change in vertical velocity from start to landing.

**中文:** 图像下的面积给出垂直速度的变化。面积 = $(-g) \times T = -gT$，等于从开始到落地垂直速度的变化。

### Exam Interpretation / 考试解读
**English:** This graph confirms that acceleration is constant throughout the motion, even at the peak. The area is used to find the change in vertical velocity.

**中文:** 此图像确认加速度在整个运动过程中恒定，即使在最高点也是如此。面积用于求垂直速度的变化。

# 8. Worked Examples / 典型例题

## Example 1: Interpreting a Vertical Velocity-Time Graph / 解读垂直速度-时间图像

### Question / 题目
**English:**
A projectile is launched vertically upward from ground level. The graph below shows its vertical velocity as a function of time. Use the graph to determine:
(a) The initial vertical velocity.
(b) The time to reach maximum height.
(c) The maximum height reached.
(d) The total time of flight.

[Assume g = 10 m s⁻² for simplicity]

**中文:**
一个抛体从地面垂直向上发射。下图显示其垂直速度随时间的变化。使用图像确定：
(a) 初始垂直速度。
(b) 达到最大高度的时间。
(c) 达到的最大高度。
(d) 总飞行时间。

[为简化，假设 g = 10 m s⁻²]

### Solution / 解答

**Step 1: Read values from the graph / 从图像读取数值**

From the graph:
- At t = 0 s, v_y = 20 m s⁻¹ → Initial vertical velocity = 20 m s⁻¹
- The line crosses the t-axis at t = 2 s → Time to maximum height = 2 s
- The line reaches v_y = -20 m s⁻¹ at t = 4 s → Total time of flight = 4 s

**Step 2: Calculate maximum height / 计算最大高度**

The maximum height is the area under the v_y-t graph from t=0 to t=2 s (the triangle above the axis):

$$ \text{Maximum height} = \frac{1}{2} \times \text{base} \times \text{height} = \frac{1}{2} \times 2 \times 20 = 20 \text{ m} $$

**Step 3: Verify using equations / 使用方程验证**

Using $v_y = u_y - gt$:
At peak, $v_y = 0$, so $0 = 20 - 10t$ → $t = 2$ s ✓

Using $s_y = u_y t - \frac{1}{2}gt^2$:
$s_y = 20(2) - \frac{1}{2}(10)(2)^2 = 40 - 20 = 20$ m ✓

### Final Answer / 最终答案
**Answer:**
(a) Initial vertical velocity = 20 m s⁻¹
(b) Time to maximum height = 2 s
(c) Maximum height = 20 m
(d) Total time of flight = 4 s

**答案：**
(a) 初始垂直速度 = 20 m s⁻¹
(b) 达到最大高度的时间 = 2 s
(c) 最大高度 = 20 m
(d) 总飞行时间 = 4 s

### Quick Tip / 提示
**English:** Always check the units on the axes before reading values. The area under a velocity-time graph gives displacement, not distance. For vertical motion, the area above the axis is positive displacement (upward), and the area below is negative displacement (downward).

**中文:** 在读取数值前始终检查坐标轴的单位。速度-时间图像下的面积给出位移，而非距离。对于垂直运动，轴上方的面积为正位移（向上），轴下方的面积为负位移（向下）。

## Example 2: Sketching Graphs from Given Data / 根据给定数据绘制图像

### Question / 题目
**English:**
A projectile is launched from ground level with an initial velocity of 30 m s⁻¹ at an angle of 30° to the horizontal. Sketch the following graphs for the motion (assume g = 10 m s⁻²):
(a) Horizontal velocity vs time
(b) Vertical velocity vs time
(c) Vertical displacement vs time

**中文:**
一个抛体从地面以30 m s⁻¹的初速度、与水平方向成30°角发射。绘制以下运动图像（假设 g = 10 m s⁻²）：
(a) 水平速度 vs 时间
(b) 垂直速度 vs 时间
(c) 垂直位移 vs 时间

### Solution / 解答

**Step 1: Calculate initial components / 计算初始分量**

$$ u_x = u \cos \theta = 30 \cos 30° = 30 \times \frac{\sqrt{3}}{2} = 15\sqrt{3} \approx 26.0 \text{ m s}^{-1} $$

$$ u_y = u \sin \theta = 30 \sin 30° = 30 \times \frac{1}{2} = 15 \text{ m s}^{-1} $$

**Step 2: Calculate key times / 计算关键时间**

Time to peak: $t_{peak} = \frac{u_y}{g} = \frac{15}{10} = 1.5 \text{ s}$

Total time of flight: $T = 2 \times t_{peak} = 3.0 \text{ s}$

**Step 3: Sketch the graphs / 绘制图像**

**(a) Horizontal velocity vs time:**
- Horizontal line at v_x = 26.0 m s⁻¹ from t=0 to t=3.0 s

**(b) Vertical velocity vs time:**
- Straight line from (0, 15) to (1.5, 0) to (3.0, -15)
- Gradient = -10 m s⁻²

**(c) Vertical displacement vs time:**
- Parabola starting at (0, 0), peaking at (1.5, h_max), ending at (3.0, 0)
- Maximum height: $h_{max} = \frac{1}{2} \times 1.5 \times 15 = 11.25 \text{ m}$

### Final Answer / 最终答案
**Answer:**
(a) Horizontal velocity-time: horizontal line at 26.0 m s⁻¹
(b) Vertical velocity-time: straight line from (0, 15) to (1.5, 0) to (3.0, -15)
(c) Vertical displacement-time: parabola with peak at (1.5, 11.25)

**答案：**
(a) 水平速度-时间：在26.0 m s⁻¹处的水平线
(b) 垂直速度-时间：从(0, 15)到(1.5, 0)到(3.0, -15)的直线
(c) 垂直位移-时间：最高点在(1.5, 11.25)的抛物线

### Quick Tip / 提示
**English:** When sketching, always label key points (start, peak, end) with their coordinates. For the vertical displacement graph, the peak occurs at the same time as the zero of the vertical velocity graph.

**中文:** 绘制时，始终用坐标标注关键点（起点、最高点、终点）。对于垂直位移图像，最高点出现在与垂直速度图像零点相同的时间。

# 10. Practical Skills Connections / 实验技能链接

**English:**
Projectile motion graphs connect to practical skills in several ways:

1. **Data Collection:** In experiments using a projectile launcher or video analysis (e.g., with a smartphone or tracker software), students collect (t, x, y) data points. Plotting these points produces the graphs studied in this sub-topic.

2. **Graph Plotting:** Students must plot displacement-time and velocity-time graphs from experimental data, including error bars. The shape of the graph (linear vs. parabolic) confirms the theoretical predictions.

3. **Gradient Calculation:** Finding the gradient of a displacement-time graph gives velocity. This is a key skill for determining initial velocity components from experimental data.

4. **Area Under Graph:** Calculating the area under a velocity-time graph from experimental data gives displacement, which can be compared to measured values.

5. **Uncertainties:** Students should estimate uncertainties in time and position measurements and represent them as error bars on graphs. The gradient uncertainty can be found using maximum and minimum slope lines.

6. **Experimental Design:** Understanding the graphs helps design experiments to measure g, initial velocity, or range. For example, the time to peak from a vertical velocity-time graph can be used to find g.

**中文:**
抛体运动图像在多个方面与实验技能相关：

1. **数据收集：** 在使用抛体发射器或视频分析（例如使用智能手机或追踪软件）的实验中，学生收集 (t, x, y) 数据点。绘制这些点产生本子知识点中研究的图像。

2. **图像绘制：** 学生必须根据实验数据绘制位移-时间和速度-时间图像，包括误差棒。图像的形状（线性 vs. 抛物线）确认理论预测。

3. **斜率计算：** 求位移-时间图像的斜率给出速度。这是从实验数据确定初始速度分量的关键技能。

4. **曲线下面积：** 从实验数据计算速度-时间图像下的面积给出位移，可以与测量值进行比较。

5. **不确定度：** 学生应估计时间和位置测量的不确定度，并在图像上用误差棒表示。斜率不确定度可以使用最大和最小斜率线找到。

6. **实验设计：** 理解图像有助于设计测量g、初速度或射程的实验。例如，从垂直速度-时间图像得到的最高点时间可用于求g。

# 12. Quick Revision Sheet / 速查表

| Category / 类别 | Key Points / 要点 |
|----------------|------------------|
| **Definition / 定义** | Projectile motion graphs show the horizontal and vertical components of displacement, velocity, and acceleration separately. The horizontal component has constant velocity (zero acceleration), while the vertical component has constant acceleration (g downward). |
| **Key Formula / 核心公式** | Horizontal: $s_x = u_x t$, $v_x = u_x$, $a_x = 0$<br>Vertical: $s_y = u_y t - \frac{1}{2}gt^2$, $v_y = u_y - gt$, $a_y = -g$ |
| **Key Graph / 核心图表** | **Vertical Velocity-Time:** Straight line with gradient -g. Crosses t-axis at peak time. Area above axis = maximum height. Area below axis = negative displacement during descent. Net area = 0 if launch and landing heights equal. |
| **Key Graph 2 / 核心图表2** | **Vertical Displacement-Time:** Parabola opening downward. Symmetric about peak. Gradient = vertical velocity (zero at peak). |
| **Key Graph 3 / 核心图表3** | **Horizontal Velocity-Time:** Horizontal line at $u \cos \theta$. Area = horizontal displacement (range). |
| **Exam Tip / 考试提示** | Always sketch horizontal and vertical graphs separately. Label axes with units. For vertical motion, remember acceleration is always downward (negative if upward is positive). The area under a velocity-time graph gives displacement, not distance. |
| **Common Mistake / 常见错误** | Confusing horizontal and vertical graphs; thinking horizontal velocity changes; forgetting that vertical acceleration is constant even at the peak; misinterpreting the area under the vertical velocity-time graph as the range. |
| **Practical Skill / 实验技能** | Use video analysis to collect (t, x, y) data. Plot graphs to verify theoretical shapes. Calculate gradient to find velocity. Calculate area under velocity-time graph to find displacement. Include error bars and uncertainty analysis. |
