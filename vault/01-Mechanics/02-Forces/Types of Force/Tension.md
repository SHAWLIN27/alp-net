# 1. Overview / 概述

**English:**
Tension is a pulling force transmitted through a string, rope, cable, or any flexible connector when it is stretched by forces acting from opposite ends. In A-Level Physics, tension is fundamental to understanding [[Types of Force]] in mechanical systems, particularly in problems involving pulleys, hanging masses, and connected objects. Tension is always a contact force that acts along the length of the connector and pulls equally on both ends (assuming a massless, inextensible string). This sub-topic is essential for [[Free-body Diagrams]] and forms the foundation for more complex mechanics problems in [[Newton's Laws of Motion]].

**中文:**
张力（Tension）是通过绳子、绳索、缆线或任何柔性连接器在两端受到相反方向拉力时传递的拉力。在A-Level物理中，张力是理解[[Types of Force]]在机械系统中作用的基础，特别是在涉及滑轮、悬挂物体和连接物体的问题中。张力始终是一种接触力，沿着连接器的长度方向作用，并对两端施加相等的拉力（假设绳子无质量且不可伸长）。本子知识点对于[[Free-body Diagrams]]至关重要，并为[[Newton's Laws of Motion]]中更复杂的力学问题奠定基础。

# 3. Core Definitions / 核心定义

| Term (EN/CN) | Definition (EN) | Definition (CN) | Common Mistakes / 常见错误 |
|--------------|-----------------|-----------------|---------------------------|

# 4. Key Concepts Explained / 关键概念详解

## 4.1 Tension as a Pulling Force / 张力作为拉力

### Explanation / 解释
**English:** Tension is fundamentally different from [[Normal Contact Force]] because it can only pull, never push. When you hold a rope and pull it, the tension force is transmitted along the rope. If you try to push a rope, it simply goes slack — tension disappears. This is why tension is always directed **away from** the object it acts on, pulling the object toward the string.

**中文:** 张力与[[Normal Contact Force]]有本质区别，因为它只能拉，不能推。当你握住绳子并拉动时，张力沿着绳子传递。如果你试图推绳子，它只会变松——张力消失。这就是为什么张力始终指向**远离**它所作用的物体，将物体拉向绳子。

### Physical Meaning / 物理意义
**English:** Tension represents the internal forces within a stretched string that resist further stretching. At the microscopic level, the atoms in the string are pulled apart slightly, creating an electromagnetic restoring force that tries to bring them back to equilibrium. This is transmitted from atom to atom along the string.

**中文:** 张力代表拉伸绳子内部抵抗进一步拉伸的内力。在微观层面，绳子中的原子被轻微拉开，产生电磁恢复力，试图将它们带回平衡位置。这种力沿着绳子从一个原子传递到另一个原子。

### Common Misconceptions / 常见误区
- **English:** Tension is NOT a force that pushes objects away. A string can only pull.
- **中文:** 张力不是推开物体的力。绳子只能拉。
- **English:** Tension does NOT act in the same direction as the string's motion — it acts along the string's length.
- **中文:** 张力不与绳子运动方向相同——它沿着绳子长度方向作用。
- **English:** In a massless string, tension is the same at both ends, even if the string goes over a frictionless pulley.
- **中文:** 在无质量绳子中，即使绳子跨过无摩擦滑轮，两端张力也相同。

### Exam Tips / 考试提示
- **English:** Always draw tension arrows pointing **away** from the object on free-body diagrams.
- **中文:** 在受力分析图上，始终将张力箭头画为**远离**物体方向。
- **English:** For connected masses, label tension as $T$ and use the same symbol for both masses if the string is massless and inextensible.
- **中文:** 对于连接物体，如果绳子无质量且不可伸长，用相同的符号 $T$ 标记两个物体的张力。

## 4.2 Tension in Pulley Systems / 滑轮系统中的张力

### Explanation / 解释
**English:** When a string passes over a [[pulley]], the tension can change direction. For an ideal pulley (frictionless and massless), the tension is the same on both sides of the pulley. This allows us to change the direction of a force without changing its magnitude. In the classic Atwood machine (two masses connected by a string over a pulley), the tension is the same for both masses, but the acceleration depends on the mass difference.

**中文:** 当绳子跨过[[pulley]]时，张力可以改变方向。对于理想滑轮（无摩擦且无质量），滑轮两侧的张力相同。这使我们能够改变力的方向而不改变其大小。在经典的阿特伍德机（两个质量通过绳子连接跨过滑轮）中，两个物体的张力相同，但加速度取决于质量差。

### Physical Meaning / 物理意义
**English:** The pulley effectively "bends" the tension force around a curve. For an ideal pulley, the tension magnitude is conserved because the pulley doesn't absorb or add energy — it simply redirects the force. Real pulleys with friction or mass will cause tension differences.

**中文:** 滑轮有效地将张力"弯曲"绕过曲线。对于理想滑轮，张力大小守恒，因为滑轮不吸收或增加能量——它只是重新定向力的方向。有摩擦或有质量的真实滑轮会导致张力差异。

### Common Misconceptions / 常见误区
- **English:** Tension is NOT always the same on both sides of a pulley — only for ideal, frictionless, massless pulleys.
- **中文:** 滑轮两侧的张力并不总是相同——仅适用于理想、无摩擦、无质量的滑轮。
- **English:** The tension in a string supporting a hanging mass is NOT equal to the weight of the mass if the system is accelerating.
- **中文:** 如果系统在加速，支撑悬挂物体的绳子中的张力不等于物体的重量。

### Exam Tips / 考试提示
- **English:** For Atwood machine problems, write separate equations for each mass using Newton's Second Law, then solve simultaneously.
- **中文:** 对于阿特伍德机问题，对每个质量分别用牛顿第二定律列方程，然后联立求解。
- **English:** Remember: $T - mg = ma$ for a mass accelerating upward, and $mg - T = ma$ for a mass accelerating downward.
- **中文:** 记住：对于向上加速的质量，$T - mg = ma$；对于向下加速的质量，$mg - T = ma$。

# 5. Essential Equations / 核心公式

## Equation 1: Tension in a Static Hanging Mass / 静态悬挂物体的张力

$$ T = mg $$

| Symbol (符号) | Meaning (EN) | Meaning (CN) | Unit (单位) |
|--------------|-------------|-------------|------------|
| $T$ | Tension in the string | 绳子中的张力 | N (Newtons) |
| $m$ | Mass of the hanging object | 悬挂物体的质量 | kg |
| $g$ | Acceleration due to gravity (9.81 m/s²) | 重力加速度 (9.81 m/s²) | m/s² |

**Conditions / 适用条件:** Object is stationary (in equilibrium). / 物体静止（处于平衡状态）。
**Limitations / 局限性:** Only applies when the object is not accelerating. / 仅适用于物体不加速的情况。

## Equation 2: Tension in an Accelerating System (Atwood Machine) / 加速系统中的张力（阿特伍德机）

$$ T = \frac{2m_1 m_2}{m_1 + m_2} g $$

| Symbol (符号) | Meaning (EN) | Meaning (CN) | Unit (单位) |
|--------------|-------------|-------------|------------|
| $T$ | Tension in the string | 绳子中的张力 | N |
| $m_1, m_2$ | Masses on either side of the pulley | 滑轮两侧的质量 | kg |
| $g$ | Acceleration due to gravity | 重力加速度 | m/s² |

**Derivation / 推导:**
For mass $m_1$ (assuming $m_1 > m_2$):
$$ m_1 g - T = m_1 a $$
For mass $m_2$:
$$ T - m_2 g = m_2 a $$
Adding both equations:
$$ m_1 g - m_2 g = (m_1 + m_2)a $$
$$ a = \frac{m_1 - m_2}{m_1 + m_2} g $$
Substituting back:
$$ T = m_2 g + m_2 a = m_2 g + m_2 \frac{m_1 - m_2}{m_1 + m_2} g = \frac{2m_1 m_2}{m_1 + m_2} g $$

**Conditions / 适用条件:** Ideal pulley (frictionless, massless), inextensible string. / 理想滑轮（无摩擦、无质量），不可伸长绳子。
**Limitations / 局限性:** Does not account for pulley mass or friction. / 不考虑滑轮质量或摩擦。

## Equation 3: Tension in a String at an Angle / 倾斜绳子中的张力

$$ T = \frac{mg}{\sin\theta} \quad \text{(for vertical support)} $$

| Symbol (符号) | Meaning (EN) | Meaning (CN) | Unit (单位) |
|--------------|-------------|-------------|------------|
| $T$ | Tension in the string | 绳子中的张力 | N |
| $m$ | Mass of the hanging object | 悬挂物体的质量 | kg |
| $\theta$ | Angle between string and horizontal | 绳子与水平方向的夹角 | degrees or radians |

**Conditions / 适用条件:** Object is in equilibrium, supported by a single string at angle $\theta$ to horizontal. / 物体处于平衡状态，由一根与水平方向成 $\theta$ 角的绳子支撑。
**Limitations / 局限性:** Only for equilibrium situations with one string. / 仅适用于一根绳子的平衡情况。

# 8. Worked Examples / 典型例题

## Example 1: Tension in a Static System / 静态系统中的张力

### Question / 题目
**English:** A 5.0 kg mass hangs stationary from a vertical string. Calculate the tension in the string. (Take $g = 9.81 \text{ m/s}^2$)

**中文:** 一个 5.0 kg 的物体静止悬挂在竖直绳子上。计算绳子中的张力。（取 $g = 9.81 \text{ m/s}^2$）

### Solution / 解答
**Step 1:** Identify forces acting on the mass.
- Weight: $mg = 5.0 \times 9.81 = 49.05 \text{ N}$ downward
- Tension: $T$ upward

**Step 2:** Apply Newton's First Law (equilibrium — stationary means $a = 0$).
$$ \Sigma F = 0 $$
$$ T - mg = 0 $$

**Step 3:** Solve for $T$.
$$ T = mg = 5.0 \times 9.81 = 49.05 \text{ N} $$

### Final Answer / 最终答案
**Answer:** $T = 49.1 \text{ N}$ (3 significant figures) | **答案：** $T = 49.1 \text{ N}$（3位有效数字）

### Quick Tip / 提示
**English:** For stationary objects, tension always equals weight. But if the object is accelerating, use Newton's Second Law instead.

**中文：** 对于静止物体，张力始终等于重力。但如果物体在加速，则使用牛顿第二定律。

---

## Example 2: Atwood Machine / 阿特伍德机

### Question / 题目
**English:** Two masses, $m_1 = 3.0 \text{ kg}$ and $m_2 = 1.0 \text{ kg}$, are connected by a light inextensible string over a frictionless pulley. Calculate:
(a) The acceleration of the system
(b) The tension in the string
(Take $g = 9.81 \text{ m/s}^2$)

**中文:** 两个质量 $m_1 = 3.0 \text{ kg}$ 和 $m_2 = 1.0 \text{ kg}$ 通过一根轻质不可伸长的绳子连接，跨过无摩擦滑轮。计算：
(a) 系统的加速度
(b) 绳子中的张力
（取 $g = 9.81 \text{ m/s}^2$）

### Solution / 解答

**Step 1:** Define the direction of motion. Since $m_1 > m_2$, $m_1$ accelerates downward, $m_2$ accelerates upward.

**Step 2:** Write Newton's Second Law for each mass.

For $m_1$ (downward positive):
$$ m_1 g - T = m_1 a $$
$$ 3.0 \times 9.81 - T = 3.0a $$
$$ 29.43 - T = 3.0a \quad \text{(1)} $$

For $m_2$ (upward positive):
$$ T - m_2 g = m_2 a $$
$$ T - 1.0 \times 9.81 = 1.0a $$
$$ T - 9.81 = a \quad \text{(2)} $$

**Step 3:** Solve simultaneously. Substitute equation (2) into (1):
$$ 29.43 - (a + 9.81) = 3.0a $$
$$ 29.43 - a - 9.81 = 3.0a $$
$$ 19.62 = 4.0a $$
$$ a = \frac{19.62}{4.0} = 4.905 \text{ m/s}^2 $$

**Step 4:** Find tension using equation (2):
$$ T = a + 9.81 = 4.905 + 9.81 = 14.715 \text{ N} $$

### Final Answer / 最终答案
**Answer:** (a) $a = 4.91 \text{ m/s}^2$ (3 s.f.) | (b) $T = 14.7 \text{ N}$ (3 s.f.)
**答案：** (a) $a = 4.91 \text{ m/s}^2$（3位有效数字）| (b) $T = 14.7 \text{ N}$（3位有效数字）

### Quick Tip / 提示
**English:** Always check: tension should be between $m_1g$ and $m_2g$. Here $9.81 < 14.7 < 29.43$, which makes sense.

**中文：** 始终检查：张力应介于 $m_1g$ 和 $m_2g$ 之间。这里 $9.81 < 14.7 < 29.43$，符合逻辑。

# 10. Practical Skills Connections / 实验技能链接

**English:**
Tension is often investigated experimentally using:
- **Spring balances / force sensors** to measure tension directly
- **Pulley systems** to study the relationship between mass and tension
- **Data loggers** to record acceleration and calculate tension indirectly

Key practical skills:
- **Measuring mass** using a balance (uncertainty ±0.01 g)
- **Measuring acceleration** using light gates or motion sensors
- **Drawing free-body diagrams** to identify forces
- **Calculating uncertainties** in tension measurements (e.g., from force sensor calibration)
- **Graph plotting:** Tension vs. acceleration to verify $T = m(g \pm a)$

**Common experiment:** Atwood machine — vary the mass difference and measure acceleration to determine $g$.

**中文：**
张力通常通过以下实验进行研究：
- **弹簧秤/力传感器** 直接测量张力
- **滑轮系统** 研究质量与张力的关系
- **数据记录器** 记录加速度并间接计算张力

关键实验技能：
- **测量质量** 使用天平（不确定度 ±0.01 g）
- **测量加速度** 使用光门或运动传感器
- **画受力分析图** 识别力
- **计算不确定度** 在张力测量中（例如来自力传感器校准）
- **绘制图表：** 张力与加速度的关系，验证 $T = m(g \pm a)$

**常见实验：** 阿特伍德机——改变质量差并测量加速度以确定 $g$。

# 12. Quick Revision Sheet / 速查表

| Category / 类别 | Key Points / 要点 |
|----------------|------------------|
| **Definition / 定义** | Tension is a pulling force transmitted through a string/cable. Always pulls, never pushes. / 张力是通过绳子/缆线传递的拉力。始终拉，从不推。 |
| **Key Formula / 核心公式** | Static: $T = mg$ | Accelerating: $T = m(g \pm a)$ | Atwood: $T = \frac{2m_1m_2}{m_1+m_2}g$ |
| **Key Graph / 核心图表** | $T$ vs $a$: Linear, gradient = mass $m$ / $T$ 与 $a$ 关系图：线性，斜率 = 质量 $m$ |
| **Direction / 方向** | Always **away** from the object on free-body diagrams / 在受力分析图上始终**远离**物体 |
| **Ideal String Assumptions / 理想绳子假设** | Massless (uniform tension) + Inextensible (constant length) / 无质量（张力均匀）+ 不可伸长（长度不变） |
| **Common Exam Setup / 常见考试设置** | Atwood machine: two masses over a frictionless pulley / 阿特伍德机：两个质量跨过无摩擦滑轮 |
| **Exam Tip / 考试提示** | For connected masses, write separate N2L equations for each mass, then solve simultaneously / 对于连接物体，对每个质量分别写牛顿第二定律方程，然后联立求解 |
| **Common Mistake / 常见错误** | Thinking tension equals weight when system is accelerating / 认为系统加速时张力等于重力 |
