# 1. Overview / 概述

**English:**
The **Definition of Work** is the foundational sub-topic within the broader [[Work Done by a Force]] topic. In physics, "work" has a very specific meaning that differs from everyday language. Work is done when a force causes an object to move through a displacement. This concept bridges [[Scalars and Vectors]] and [[Newton's Laws of Motion]], forming the basis for understanding energy transfer. Without a clear grasp of this definition, students cannot progress to [[Work Done by a Force at an Angle]] or [[Force-Displacement Graphs]].

**中文:**
**功的定义** 是 [[Work Done by a Force]] 主题下的基础子知识点。在物理学中，"功"有着与日常用语截然不同的特定含义。当力使物体发生位移时，才算是做了功。这个概念连接了 [[Scalars and Vectors]] 和 [[Newton's Laws of Motion]]，是理解能量传递的基础。如果没有清晰掌握这个定义，学生就无法继续学习 [[Work Done by a Force at an Angle]] 或 [[Force-Displacement Graphs]]。

# 2. Core Definition / 核心定义

| Term (EN/CN) | Definition (EN) | Definition (CN) |
| ------------ | --------------- | --------------- |
| **Work** / 功 | The product of the force applied to an object and the displacement of the object in the direction of the force. | 作用在物体上的力与物体在力的方向上发生的位移的乘积。 |
| **Displacement** / 位移 | The straight-line distance from the starting point to the ending point, with direction. | 从起点到终点的直线距离，具有方向性。 |
| **Force** / 力 | A push or pull that can change the motion of an object. | 能够改变物体运动状态的推或拉。 |
| **Joule (J)** / 焦耳 | The SI unit of work; 1 J = 1 N·m. | 功的国际单位；1 J = 1 N·m。 |
| **Scalar** / 标量 | A quantity that has magnitude only, no direction. Work is a scalar. | 只有大小没有方向的量。功是标量。 |
| **Energy Transfer** / 能量传递 | Work done on an object equals the energy transferred to that object. | 对物体做的功等于传递给该物体的能量。 |

# 3. Key Concepts / 关键概念

**English:**
The definition of work is deceptively simple: $W = F \cdot s$, where $F$ is the force and $s$ is the displacement. However, several critical points must be understood:

1. **Force and displacement must be in the same direction.** If you push a wall and it doesn't move ($s = 0$), no work is done. This is a common exam trap.

2. **Work is a scalar quantity.** Despite involving force (a vector) and displacement (a vector), work itself has no direction. This is because work is the dot product of two vectors: $W = \vec{F} \cdot \vec{s}$.

3. **Work represents energy transfer.** When work is done on an object, its energy changes. This connects directly to [[Kinetic Energy and Potential Energy]].

4. **Only the component of force in the direction of displacement does work.** This leads naturally to the [[Work Done by a Force at an Angle]] sub-topic.

**Common Pitfall:** Students often think that holding a heavy object stationary requires work. In physics, if there is no displacement, no work is done — even if you feel tired! The tiredness comes from internal muscle contractions doing work on your own body, not on the object.

**中文:**
功的定义看似简单：$W = F \cdot s$，其中 $F$ 是力，$s$ 是位移。但有几个关键点必须理解：

1. **力和位移必须在同一方向上。** 如果你推一堵墙而墙没有移动（$s = 0$），则没有做功。这是一个常见的考试陷阱。

2. **功是标量。** 尽管涉及力（矢量）和位移（矢量），但功本身没有方向。这是因为功是两个矢量的点积：$W = \vec{F} \cdot \vec{s}$。

3. **功代表能量传递。** 当对物体做功时，其能量会发生变化。这直接连接到 [[Kinetic Energy and Potential Energy]]。

4. **只有位移方向上的力分量才做功。** 这自然引出了 [[Work Done by a Force at an Angle]] 子知识点。

**常见误区：** 学生常常认为举起一个重物保持不动需要做功。在物理学中，如果没有位移，就没有做功——即使你感到疲劳！疲劳来自于内部肌肉收缩对你自己的身体做功，而不是对物体做功。

# 4. Formulas / 公式

The fundamental formula for work when force and displacement are in the same direction:

$$ W = F \cdot s $$

| Symbol (符号) | Meaning (EN) | Meaning (CN) | Unit (单位) |
| ------------- | ------------ | ------------ | ----------- |
| $W$ | Work done | 做功 | J (joule / 焦耳) |
| $F$ | Force applied | 施加的力 | N (newton / 牛顿) |
| $s$ | Displacement in direction of force | 力方向上的位移 | m (metre / 米) |

**Derivation / 推导:**
This formula is derived from the definition: work is the product of force and displacement in the direction of the force. It can be derived from the more general dot product form: $W = \vec{F} \cdot \vec{s} = Fs\cos\theta$, where $\theta = 0^\circ$ (force and displacement in same direction), so $\cos 0^\circ = 1$, giving $W = Fs$.

**Conditions / 适用条件:**
- Force must be constant (not varying with position or time)
- Force and displacement must be in the same direction
- For varying forces, use [[Force-Displacement Graphs]]

# 6. Worked Example / 典型例题

### Example 1: Basic Work Calculation

### Question / 题目
**English:** A student pushes a box with a constant force of 50 N across a horizontal floor. The box moves 3.0 m in the direction of the force. Calculate the work done by the student.

**中文:** 一名学生用50 N的恒力在水平地板上推一个箱子。箱子沿力的方向移动了3.0 m。计算该学生所做的功。

### Solution / 解答
**Step 1:** Identify the known quantities.
$F = 50 \text{ N}$, $s = 3.0 \text{ m}$

**Step 2:** Apply the work formula.
Since force and displacement are in the same direction:
$$W = F \cdot s$$

**Step 3:** Substitute values.
$$W = 50 \times 3.0 = 150 \text{ J}$$

### Final Answer / 最终答案
**Answer:** 150 J **答案:** 150 焦耳

### Quick Tip / 提示
Always check that the force and displacement are in the same direction before using $W = Fs$. If they are not, you need the [[Work Done by a Force at an Angle]] formula.

---

### Example 2: Zero Work Scenario

### Question / 题目
**English:** A weightlifter holds a 200 kg barbell stationary above his head for 5 seconds. How much work is done on the barbell during this time? (Take $g = 9.8 \text{ m/s}^2$)

**中文:** 一名举重运动员将200 kg的杠铃静止举过头顶5秒钟。在这段时间内，对杠铃做了多少功？（取 $g = 9.8 \text{ m/s}^2$）

### Solution / 解答
**Step 1:** Identify the situation.
The barbell is stationary, so displacement $s = 0 \text{ m}$.

**Step 2:** Apply the work formula.
$$W = F \cdot s = F \times 0 = 0 \text{ J}$$

**Note:** The force applied by the weightlifter is $F = mg = 200 \times 9.8 = 1960 \text{ N}$, but since there is no displacement, no work is done on the barbell.

### Final Answer / 最终答案
**Answer:** 0 J **答案:** 0 焦耳

### Quick Tip / 提示
This is a classic exam question. Remember: no displacement = no work, regardless of how much force is applied or how tired the person feels!
