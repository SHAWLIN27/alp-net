# 1. Overview / 概述

**English:**
This sub-topic focuses on the formation and properties of stationary (standing) waves on a stretched string. When two progressive waves of the same frequency and amplitude travel in opposite directions along a string, they [[Superposition and Interference|superpose]] to form a stationary wave pattern. The string is fixed at both ends, creating boundary conditions that determine which wavelengths can exist. This leads to the concept of **harmonics** — the fundamental frequency and its overtones. Understanding stationary waves on strings is essential for musical instruments (guitars, violins, pianos) and for understanding [[Stationary Waves in Pipes (Open and Closed)|stationary waves in pipes]]. This sub-topic builds directly on [[Formation of Stationary Waves]] and [[Nodes and Antinodes]].

**中文:**
本子知识点聚焦于拉伸弦线上驻波的形成与特性。当两列频率和振幅相同的行波沿弦线相向传播时，它们通过[[Superposition and Interference|叠加]]形成驻波图样。弦线两端固定，产生边界条件，决定了哪些波长可以存在。这引出了**谐波**的概念——基频及其泛音。理解弦线上的驻波对于乐器（吉他、小提琴、钢琴）以及理解[[Stationary Waves in Pipes (Open and Closed)|管中的驻波]]至关重要。本子知识点直接建立在[[Formation of Stationary Waves]]和[[Nodes and Antinodes]]之上。

# 3. Core Definitions / 核心定义

| Term (EN/CN) | Definition (EN) | Definition (CN) | Common Mistakes / 常见错误 |
|--------------|-----------------|-----------------|---------------------------|
| **Stationary Wave** / 驻波 | A wave formed by the superposition of two progressive waves of the same frequency and amplitude traveling in opposite directions, resulting in a pattern of nodes and antinodes that does not propagate. | 由两列频率和振幅相同、相向传播的行波叠加形成的波，产生不传播的波节和波腹图样。 | ❌ Confusing stationary waves with progressive waves — stationary waves do NOT transfer energy. |
| **Node** / 波节 | A point on a stationary wave where the displacement is always zero due to destructive interference. | 驻波上因相消干涉而位移始终为零的点。 | ❌ Thinking nodes have maximum displacement. |
| **Antinode** / 波腹 | A point on a stationary wave where the displacement varies between maximum positive and maximum negative values due to constructive interference. | 驻波上因相长干涉而位移在最大正值和最大负值之间变化的点。 | ❌ Thinking antinodes are stationary. |
| **Fundamental Mode** / 基模 | The lowest frequency mode of vibration of a stationary wave system, corresponding to the simplest standing wave pattern (one loop). | 驻波系统的最低频率振动模式，对应最简单的驻波图样（一个波腹）。 | ❌ Confusing fundamental with first harmonic (they are the same). |
| **Harmonic** / 谐波 | A component frequency of a stationary wave that is an integer multiple of the fundamental frequency. | 驻波中为基频整数倍的分量频率。 | ❌ Thinking all harmonics are present in all systems. |
| **Tension ($T$)** / 张力 | The force applied to stretch a string, which affects the wave speed. | 施加在弦线上使其拉伸的力，影响波速。 | ❌ Confusing tension with mass per unit length. |

# 4. Key Concepts Explained / 关键概念详解

## 4.1 Formation of Stationary Waves on a String / 弦线上驻波的形成

### Explanation / 解释
**English:** When a string fixed at both ends is plucked or vibrated, a progressive wave travels along the string. At the fixed end, the wave reflects. The reflected wave travels back along the string, superposing with the incident wave. Since both ends are fixed, they must be [[Nodes and Antinodes|nodes]] — points of zero displacement. This boundary condition restricts the possible wavelengths to those where the string length $L$ equals an integer number of half-wavelengths: $L = n\frac{\lambda}{2}$, where $n = 1, 2, 3, ...$ is the harmonic number.

**中文:** 当两端固定的弦线被拨动或振动时，行波沿弦线传播。在固定端，波发生反射。反射波沿弦线返回，与入射波叠加。由于两端固定，它们必须是[[Nodes and Antinodes|波节]]——位移为零的点。这一边界条件将可能的波长限制为弦长 $L$ 等于半波长整数倍的情况：$L = n\frac{\lambda}{2}$，其中 $n = 1, 2, 3, ...$ 是谐波数。

### Physical Meaning / 物理意义
**English:** The fixed ends cannot move, so they must always be nodes. This creates a "standing" pattern where energy is trapped between the two ends. The string vibrates in segments called **loops**, each loop being half a wavelength long. The number of loops equals the harmonic number $n$.

**中文:** 固定端不能移动，因此它们必须是波节。这产生了一个"驻立"的图样，能量被束缚在两端之间。弦线以称为**波腹段**的片段振动，每个波腹段长度为半个波长。波腹段的数量等于谐波数 $n$。

### Common Misconceptions / 常见误区
- ❌ **English:** "The wave travels along the string." → No, the pattern is stationary; only the individual particles vibrate transversely.
- ❌ **中文:** "波沿弦线传播。" → 不对，图样是驻立的；只有单个粒子在横向振动。
- ❌ **English:** "All points on the string have the same amplitude." → No, amplitude varies from zero at nodes to maximum at antinodes.
- ❌ **中文:** "弦线上所有点的振幅相同。" → 不对，振幅从波节处的零变化到波腹处的最大值。
- ❌ **English:** "The string is straight at all times." → No, at antinodes the displacement is maximum at certain instants.
- ❌ **中文:** "弦线始终是直的。" → 不对，在波腹处，位移在某些时刻达到最大值。

### Exam Tips / 考试提示
- ✅ **English:** Always draw the stationary wave pattern with nodes clearly marked at the fixed ends. For the fundamental mode, there is exactly one antinode in the middle.
- ✅ **中文:** 始终在固定端清晰标记波节来绘制驻波图样。对于基模，中间恰好有一个波腹。
- ✅ **English:** Remember: $L = n\frac{\lambda}{2}$ is the key relationship — use it to find wavelength from string length.
- ✅ **中文:** 记住：$L = n\frac{\lambda}{2}$ 是关键关系——用它从弦长求波长。

## 4.2 Harmonics and Overtones / 谐波与泛音

### Explanation / 解释
**English:** The **fundamental mode** ($n=1$) has the lowest frequency and produces the simplest pattern: one loop with nodes at both ends and one antinode in the middle. The **second harmonic** ($n=2$) has two loops, with a node at the center. The **third harmonic** ($n=3$) has three loops, and so on. The frequencies of the harmonics are integer multiples of the fundamental frequency: $f_n = n f_1$.

**中文:** **基模** ($n=1$) 具有最低频率，产生最简单的图样：一个波腹段，两端为波节，中间为一个波腹。**二次谐波** ($n=2$) 有两个波腹段，中心有一个波节。**三次谐波** ($n=3$) 有三个波腹段，依此类推。谐波的频率是基频的整数倍：$f_n = n f_1$。

### Physical Meaning / 物理意义
**English:** The harmonic series determines the **timbre** (tone quality) of a musical instrument. Different instruments produce different mixtures of harmonics, which is why a guitar sounds different from a violin even when playing the same note.

**中文:** 谐波序列决定了乐器的**音色**（音质）。不同的乐器产生不同的谐波混合，这就是为什么即使演奏同一个音符，吉他的声音也与小提琴不同。

### Common Misconceptions / 常见误区
- ❌ **English:** "The fundamental frequency is the first harmonic." → Actually, they are the same thing. The fundamental is the first harmonic ($n=1$).
- ❌ **中文:** "基频就是一次谐波。" → 实际上，它们是同一个东西。基频就是一次谐波 ($n=1$)。
- ❌ **English:** "All harmonics are equally loud." → No, the amplitude of each harmonic depends on how the string is excited.
- ❌ **中文:** "所有谐波一样响亮。" → 不对，每个谐波的振幅取决于弦线被激发的方式。

### Exam Tips / 考试提示
- ✅ **English:** For a string fixed at both ends, only **integer harmonics** are possible ($n=1, 2, 3, ...$). This is different from pipes.
- ✅ **中文:** 对于两端固定的弦线，只有**整数谐波**是可能的 ($n=1, 2, 3, ...$)。这与管不同。
- ✅ **English:** The number of antinodes equals $n$, and the number of nodes equals $n+1$ (including the fixed ends).
- ✅ **中文:** 波腹的数量等于 $n$，波节的数量等于 $n+1$（包括固定端）。

## 4.3 Wave Speed on a String / 弦线上的波速

### Explanation / 解释
**English:** The speed of a transverse wave on a stretched string depends on two factors: the **tension** $T$ in the string and the **mass per unit length** $\mu$ (also called linear density). The relationship is:

$$ v = \sqrt{\frac{T}{\mu}} $$

where $T$ is in newtons (N) and $\mu$ is in kg m$^{-1}$.

**中文:** 拉伸弦线上横波的波速取决于两个因素：弦线的**张力** $T$ 和**单位长度质量** $\mu$（也称为线密度）。关系为：

$$ v = \sqrt{\frac{T}{\mu}} $$

其中 $T$ 的单位为牛顿 (N)，$\mu$ 的单位为 kg m$^{-1}$。

### Physical Meaning / 物理意义
**English:** Increasing tension increases wave speed (tightening a guitar string raises the pitch). Increasing mass per unit length decreases wave speed (thicker strings produce lower notes).

**中文:** 增加张力会提高波速（拧紧吉他弦会提高音调）。增加单位长度质量会降低波速（较粗的弦产生较低的音符）。

### Common Misconceptions / 常见误区
- ❌ **English:** "Wave speed depends on frequency." → No, for a given string under fixed tension, wave speed is constant for all frequencies.
- ❌ **中文:** "波速取决于频率。" → 不对，对于给定张力下的弦线，所有频率的波速都是恒定的。
- ❌ **English:** "Mass per unit length is the same as density." → No, $\mu$ is mass per unit length (kg/m), not mass per unit volume (kg/m$^3$).
- ❌ **中文:** "单位长度质量就是密度。" → 不对，$\mu$ 是单位长度质量 (kg/m)，不是单位体积质量 (kg/m$^3$)。

### Exam Tips / 考试提示
- ✅ **English:** Use $v = f\lambda$ together with $v = \sqrt{T/\mu}$ to derive the frequency equation.
- ✅ **中文:** 将 $v = f\lambda$ 与 $v = \sqrt{T/\mu}$ 结合使用来推导频率方程。
- ✅ **English:** Remember: $\mu = \frac{\text{mass of string}}{\text{length of string}}$.
- ✅ **中文:** 记住：$\mu = \frac{\text{弦线质量}}{\text{弦线长度}}$。

# 5. Essential Equations / 核心公式

## Equation 1: String Length and Wavelength / 弦长与波长

$$ L = n\frac{\lambda}{2} $$

| Symbol (符号) | Meaning (EN) | Meaning (CN) | Unit (单位) |
|--------------|-------------|-------------|------------|
| $L$ | Length of string | 弦线长度 | m |
| $n$ | Harmonic number (1, 2, 3, ...) | 谐波数 (1, 2, 3, ...) | — |
| $\lambda$ | Wavelength | 波长 | m |

**Derivation / 推导:** For a string fixed at both ends, each loop is half a wavelength. The number of loops equals $n$, so total length $L = n \times (\lambda/2)$.

**Conditions / 适用条件:** String fixed at both ends. | 两端固定的弦线。

**Limitations / 局限性:** Only applies to ideal strings with no stiffness. Real strings have slight stiffness that affects higher harmonics. | 仅适用于无刚度的理想弦线。实际弦线的微小刚度会影响高次谐波。

## Equation 2: Wave Speed on a String / 弦线上的波速

$$ v = \sqrt{\frac{T}{\mu}} $$

| Symbol (符号) | Meaning (EN) | Meaning (CN) | Unit (单位) |
|--------------|-------------|-------------|------------|
| $v$ | Wave speed | 波速 | m s$^{-1}$ |
| $T$ | Tension in the string | 弦线张力 | N |
| $\mu$ | Mass per unit length | 单位长度质量 | kg m$^{-1}$ |

**Derivation / 推导:** Derived from wave equation on a string under tension. Not required at A-Level.

**Conditions / 适用条件:** String under tension, transverse waves. | 拉伸弦线，横波。

**Limitations / 局限性:** Assumes string is perfectly flexible and uniform. | 假设弦线完全柔韧且均匀。

## Equation 3: Frequency of Harmonics / 谐波频率

$$ f_n = \frac{n}{2L}\sqrt{\frac{T}{\mu}} $$

| Symbol (符号) | Meaning (EN) | Meaning (CN) | Unit (单位) |
|--------------|-------------|-------------|------------|
| $f_n$ | Frequency of the $n$th harmonic | 第 $n$ 次谐波的频率 | Hz |
| $n$ | Harmonic number | 谐波数 | — |
| $L$ | Length of string | 弦线长度 | m |
| $T$ | Tension | 张力 | N |
| $\mu$ | Mass per unit length | 单位长度质量 | kg m$^{-1}$ |

**Derivation / 推导:** From $v = f\lambda$ and $v = \sqrt{T/\mu}$ and $\lambda = 2L/n$:
$$ f = \frac{v}{\lambda} = \frac{\sqrt{T/\mu}}{2L/n} = \frac{n}{2L}\sqrt{\frac{T}{\mu}} $$

**Conditions / 适用条件:** String fixed at both ends, transverse stationary waves. | 两端固定的弦线，横驻波。

**Limitations / 局限性:** Assumes ideal string. Real strings may have slightly different frequencies due to stiffness. | 假设理想弦线。实际弦线因刚度可能频率略有不同。

# 8. Worked Examples / 典型例题

## Example 1: Finding Frequency of a Harmonic / 求谐波频率

### Question / 题目
**English:** A string of length 0.80 m and mass 4.0 g is stretched with a tension of 50 N. Calculate:
(a) The mass per unit length of the string.
(b) The speed of waves on the string.
(c) The fundamental frequency of the string.
(d) The frequency of the third harmonic.

**中文:** 一根长度为 0.80 m、质量为 4.0 g 的弦线以 50 N 的张力拉伸。计算：
(a) 弦线的单位长度质量。
(b) 弦线上的波速。
(c) 弦线的基频。
(d) 三次谐波的频率。

### Solution / 解答

**(a) Mass per unit length / 单位长度质量**

$$ \mu = \frac{m}{L} = \frac{4.0 \times 10^{-3} \text{ kg}}{0.80 \text{ m}} = 5.0 \times 10^{-3} \text{ kg m}^{-1} $$

**(b) Wave speed / 波速**

$$ v = \sqrt{\frac{T}{\mu}} = \sqrt{\frac{50}{5.0 \times 10^{-3}}} = \sqrt{10000} = 100 \text{ m s}^{-1} $$

**(c) Fundamental frequency / 基频**

For $n=1$: $\lambda_1 = 2L = 2 \times 0.80 = 1.60 \text{ m}$

$$ f_1 = \frac{v}{\lambda_1} = \frac{100}{1.60} = 62.5 \text{ Hz} $$

Alternatively: $f_1 = \frac{1}{2L}\sqrt{\frac{T}{\mu}} = \frac{1}{2 \times 0.80}\sqrt{\frac{50}{5.0 \times 10^{-3}}} = \frac{100}{1.60} = 62.5 \text{ Hz}$

**(d) Third harmonic frequency / 三次谐波频率**

$$ f_3 = 3f_1 = 3 \times 62.5 = 187.5 \text{ Hz} $$

### Final Answer / 最终答案
**Answer:** (a) $\mu = 5.0 \times 10^{-3} \text{ kg m}^{-1}$, (b) $v = 100 \text{ m s}^{-1}$, (c) $f_1 = 62.5 \text{ Hz}$, (d) $f_3 = 187.5 \text{ Hz}$ | **答案：** (a) $\mu = 5.0 \times 10^{-3} \text{ kg m}^{-1}$, (b) $v = 100 \text{ m s}^{-1}$, (c) $f_1 = 62.5 \text{ Hz}$, (d) $f_3 = 187.5 \text{ Hz}$

### Quick Tip / 提示
**English:** Always convert mass to kg and length to m before calculating $\mu$. Check that your units are consistent — tension in N, $\mu$ in kg/m gives speed in m/s.

**中文:** 在计算 $\mu$ 之前，始终将质量转换为 kg，长度转换为 m。检查单位是否一致——张力用 N，$\mu$ 用 kg/m，得到的速度单位为 m/s。

## Example 2: Finding Tension from Frequency / 从频率求张力

### Question / 题目
**English:** A guitar string of length 0.65 m and mass per unit length $4.0 \times 10^{-3} \text{ kg m}^{-1}$ is tuned to produce a fundamental frequency of 220 Hz. Calculate the tension required in the string.

**中文:** 一根长度为 0.65 m、单位长度质量为 $4.0 \times 10^{-3} \text{ kg m}^{-1}$ 的吉他弦被调音以产生 220 Hz 的基频。计算弦线所需的张力。

### Solution / 解答

Use the fundamental frequency equation for $n=1$:

$$ f_1 = \frac{1}{2L}\sqrt{\frac{T}{\mu}} $$

Rearrange to find $T$:

$$ 2L f_1 = \sqrt{\frac{T}{\mu}} $$

$$ (2L f_1)^2 = \frac{T}{\mu} $$

$$ T = \mu (2L f_1)^2 $$

Substitute values:

$$ T = (4.0 \times 10^{-3}) \times (2 \times 0.65 \times 220)^2 $$

$$ T = (4.0 \times 10^{-3}) \times (286)^2 $$

$$ T = (4.0 \times 10^{-3}) \times 81796 $$

$$ T = 327 \text{ N} $$

### Final Answer / 最终答案
**Answer:** $T = 327 \text{ N}$ | **答案：** $T = 327 \text{ N}$

### Quick Tip / 提示
**English:** When rearranging, be careful with the square. Remember: $(2L f_1)^2 = 4L^2 f_1^2$. Always check that your answer is reasonable — guitar strings typically have tensions of 50-150 N, so 327 N is high but possible for a steel string.

**中文:** 重新排列时，注意平方。记住：$(2L f_1)^2 = 4L^2 f_1^2$。始终检查答案是否合理——吉他弦的张力通常在 50-150 N 之间，所以 327 N 较高，但对于钢弦是可能的。

# 10. Practical Skills Connections / 实验技能链接

**English:** This sub-topic connects to practical work in several ways:

1. **Determining the speed of sound in air** using stationary waves (resonance tube experiment) — though this is more relevant to [[Stationary Waves in Pipes (Open and Closed)]], the principles are the same.

2. **Verifying the relationship $f \propto \sqrt{T}$** : Use the experimental setup shown in Section 7.2. Vary the hanging mass (changing tension) and measure the fundamental frequency using a signal generator. Plot $f$ against $\sqrt{T}$ — should be a straight line through the origin.

3. **Measuring mass per unit length** : Weigh a known length of string to find $\mu$. This involves using a balance (measurement uncertainty ±0.01 g) and a ruler (±1 mm).

4. **Uncertainties** : When calculating $f$ from $L$, $T$, and $\mu$, propagate uncertainties using:
   - $\frac{\Delta f}{f} = \frac{\Delta L}{L} + \frac{1}{2}\frac{\Delta T}{T} + \frac{1}{2}\frac{\Delta \mu}{\mu}$

5. **Graph plotting** : Plot $f$ vs $n$ to find fundamental frequency from gradient. Plot $f$ vs $\sqrt{T}$ to verify proportionality.

**中文:** 本子知识点通过以下几种方式与实验工作联系：

1. **测定空气中的声速** 使用驻波（共振管实验）——虽然这与[[Stationary Waves in Pipes (Open and Closed)]]更相关，但原理相同。

2. **验证 $f \propto \sqrt{T}$ 的关系**：使用第 7.2 节所示的实验装置。改变悬挂质量（改变张力）并使用信号发生器测量基频。绘制 $f$ 对 $\sqrt{T}$ 的图——应为通过原点的直线。

3. **测量单位长度质量**：称量已知长度的弦线以求出 $\mu$。这涉及使用天平（测量不确定度 ±0.01 g）和尺子（±1 mm）。

4. **不确定度**：当从 $L$、$T$ 和 $\mu$ 计算 $f$ 时，使用以下方法传播不确定度：
   - $\frac{\Delta f}{f} = \frac{\Delta L}{L} + \frac{1}{2}\frac{\Delta T}{T} + \frac{1}{2}\frac{\Delta \mu}{\mu}$

5. **绘图**：绘制 $f$ 对 $n$ 的图以从斜率求基频。绘制 $f$ 对 $\sqrt{T}$ 的图以验证比例关系。

# 12. Quick Revision Sheet / 速查表

| Category / 类别 | Key Points / 要点 |
|----------------|------------------|
| **Definition / 定义** | Stationary wave on a string: formed by superposition of two progressive waves of same $f$ and $A$ traveling in opposite directions. Fixed ends are always nodes. |
| **Key Formula / 核心公式** | $L = n\frac{\lambda}{2}$ (wavelength condition) |
| | $v = \sqrt{\frac{T}{\mu}}$ (wave speed) |
| | $f_n = \frac{n}{2L}\sqrt{\frac{T}{\mu}}$ (frequency of $n$th harmonic) |
| **Key Graph / 核心图表** | $f$ vs $n$: straight line through origin, gradient = $f_1$ |
| | $f$ vs $\sqrt{T}$: straight line through origin (for fixed $L$ and $\mu$) |
| **Key Diagram / 核心图表** | First three harmonics: $n=1$ (1 loop, 1 antinode), $n=2$ (2 loops, 1 central node), $n=3$ (3 loops, 2 internal nodes) |
| **Common Exam Question / 常见考题** | "Calculate the fundamental frequency of a string of length $L$, mass $m$, under tension $T$." |
| **Common Mistake / 常见错误** | ❌ Forgetting that $L = n\lambda/2$, not $L = n\lambda$ |
| | ❌ Using mass instead of mass per unit length $\mu$ |
| | ❌ Confusing nodes and antinodes |
| **Exam Tip / 考试提示** | ✅ Always draw the stationary wave pattern first — it helps visualize the problem |
| | ✅ Check units: $T$ in N, $\mu$ in kg/m, $L$ in m |
| | ✅ For $n=1$, $\lambda = 2L$; for $n=2$, $\lambda = L$; for $n=3$, $\lambda = 2L/3$ |
| **Practical Tip / 实验提示** | In the experiment, adjust the frequency until resonance occurs (maximum amplitude). The vibrating length $L$ is measured between the vibration generator and the pulley/bridge. |
| **CIE Only / CIE 特有** | CIE 9702 requires graphical method for formation of stationary waves (8.2a) |
| **Edexcel Only / Edexcel 特有** | Edexcel IAL requires derivation of $f = \frac{n}{2L}\sqrt{\frac{T}{\mu}}$ (5.19) |

> 📋 **CIE Only:** CIE 9702 8.2(a) specifically requires explaining the formation of a stationary wave using a **graphical method** — you should be able to draw the incident and reflected waves and show how they superpose to form nodes and antinodes.

> 📋 **Edexcel Only:** Edexcel IAL 5.19 requires you to **derive** the frequency equation $f = \frac{n}{2L}\sqrt{\frac{T}{\mu}}$ from $v = f\lambda$ and $v = \sqrt{T/\mu}$. Make sure you can show this step-by-step.
